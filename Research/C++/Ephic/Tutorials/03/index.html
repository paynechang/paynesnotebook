Tutorial #3

  • Command waveform: Alpha
  • Command waveform: Chirp
  • Command waveform: Ramp
  • Command waveform: Sine

In this tutorial, Ephic is in the I Clamp mode.

Command Waveform: Alpha

The peak of the alpha function is set to 0.2 nA. The waveform starts at 4 ms. The time to the peak is 2 ms.

Command Waveform: Chirp

In this example, the amplitude of chirp is set to 0.2 nA. The waveform starts at 1000 ms. The length of chirp is 10 seconds (10000 ms). The frequency increases from 0 to 10 Hz. Thus, the frequency increases at the rate of 1 Hz per second.

Command Waveform: Ramp

The amplitude of this ramp is set to 0.4 nA. It starts at 4 ms. The length of it is 12 ms.

Command Waveform: Sine

The amplitude of this sine wave is set to 0.5 nA. It starts at 10 ms. The length of it is 30 ms. The frequency is 100 Hz. Thus, the period is 10 ms and there are three cycles in 30 ms.
