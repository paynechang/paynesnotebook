NEURON Tutorial #9

Goals: Measure Rin and rebound slope through linear regression

Downloads:

  • Main.hoc - Program start here
  • ExperimentControl.hoc - Experiment contols
  • ParameterControl.hoc - Parameter controls
  • RunControl.hoc - Run controls
  • Graph.hoc - A window contains a voltage graph and a current graph
  • FittingWindow.hoc - For linear regression analysis
  • 1_Pulse.hoc - For pulse experiment (membrane time connstant)
  • 2_Step.hoc - For step experiment (Rin, sag, rebound slope)
  • 3_Alpha.hoc - For alpha experiment ( alpha summation)
  • 09.zip - Comprises all files.

Steps:

  • Double click "Main.hoc"
  • Click "Current Step (500 ms)" button.
  • Click "Run Step Family" button.

  • Select "Fit to Data" / "Common Functional Forms" / "Linear"
  • Select "Fit to Data" / "Read Data File".
  • Select "v_i.dat"
  • Check "Watch the fit" box.
  • Click "Praxis fit" button.
    Slope = 52.1 (MΩ)

  • Select "Fit to data" / "Read Data File".
  • Select "rebound_i.dat"
  • Click "Praxis fit" button.
    Rebound slope = -0.23 (mV/mV)
