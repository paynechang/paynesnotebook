NEURON Tutorial #13

Goals: Develop an integrated environment to test one Ih model.

Specific Aims:

  • Develop an integrated environment for the following measurements.
    • Membrane time constant.
    • Input resistance.
    • Rebound slope.
    • Temporal summation.
    • Impedance magnitude profile.
    • Voltage lag (phase).
    • GV curve.
    • Activation time constant.
    • Deactivation time constant.
  • Test Sonia's Ih model (J.Neurosci. 24:11046-11056 (2004)).

Downloads:

  • Alpha.hoc - For temporal summation experiment (alpha EPSCs)
  • Chirp.hoc - For chirp experiment
  • ExperimentControl.hoc - Experiment contols
  • Fitting.hoc - Procedures for curve fitting
  • GraphVI.hoc - Voltage and current graphs
  • Main.hoc - Program start here
  • ParameterControl.hoc - Parameter controls
  • Pulse.hoc - For pulse experiment (membrane time connstant)
  • RunControl.hoc - Run controls
  • Step.hoc - For current step experiment (Rin, sag, rebound slope)
  • VoltageClamp.hoc - Voltage clamp experiment
  • 13.zip - Contains all files.

With this set of parameters, the resting membrane potential is hyperpolarized. To have a resting potential at -65 mV, change gbar_h to 4e-5 and you will have the resting membrane potential at -65 mV.
